Question-1: What are the different ways to select an element in the DOM?

getElementById("id"),
getElementsByClassName("class"),
getElementsByTagName("tag"),
querySelector("CSS selector"),
querySelectorAll("CSS selector").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML → Use when dealing with HTML elements.
innerText → Use when you need only visible text.
textContent → Use when you need all text, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of adding event listeners to multiple child elements, we add a single event listener to a parent element and use event bubbling to handle events on child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event, such as a click or keypress, propagates from the target element up to its ancestors (parent elements), all the way to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement(tagName)
Add: parent.appendChild(child) or parent.insertBefore(newElement, referenceNode)
Remove: parent.removeChild(child) or element.remove()
